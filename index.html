Sales signal by game

A prediction of whether marketplace sales are likely to spike or dip, based on the high- and medium-impact news in the selected window.

How the prediction works: new banners & version updates (High impact) push demand up → spike likely; events & cosmetics (Medium) give a mild lift; maintenance/downtime (None) can cause a short dip; esports/dev news barely moves sales.
To validate against real numbers: drop a sales.json next to this file shaped like [{"game":"Genshin Impact","date":"2026-06-01","value":1234}] and we'll compare predicted vs. actual and flag anomalies. Keep real sales private — never on the public Pages URL.

Where the news comes from

This dashboard reads a file called data.json, which GitHub refreshes automatically on a schedule. You normally never need to touch this. Use the box only if you want to test a different data file.

Valorant preview key optional

Only needed if you want the Preview button in the Sources tab to pull Valorant news instantly in your browser. The automatic scheduled pull already uses the key you saved in GitHub, so most people can ignore this.

Smarter categorization coming later

Right now, news is sorted into categories using simple keyword rules. Later, we can let Claude (AI) read each headline and judge its type and sales impact far more accurately — even in other languages. This isn't switched on yet; when you're ready, add an Anthropic key in GitHub and flip one setting.

How it all works

1. Collect — GitHub automatically pulls official news from each game on a timer. 2. Categorize — each item is tagged (Banner / Update / Event / Maintenance / Esports / Cosmetic / Other) with a likely sales impact. 3. Show — it appears here in the dashboard. 4. Predict — the Sales Impact tab turns high-impact news into a spike/dip signal. No Google Sheet and no server to maintain.
